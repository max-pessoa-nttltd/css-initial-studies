O poder do CSS

Estilos, animações e recursos sofisticados — vistos ao vivo, com o código real ao lado. Tudo que você rolar daqui pra baixo é feito só com HTML e CSS. Sem frameworks, sem mágica.

role para começar
01

Cores & gradientes

Antes de animar, a gente pinta. CSS combina cores sólidas (hex, rgb, hsl) com gradientes lineares, radiais e cônicos para criar profundidade sem usar uma única imagem.

Resultado
linear
radial
conic
hsl()
CSS
.linear {
  background: linear-gradient(135deg, #FF6B6B, #FFC857);
}
.radial {
  background: radial-gradient(circle, #4ECDC4, #2C2558);
}
.conic {
  background: conic-gradient(#FF6B6B, #FFC857, #4ECDC4);
}
.hsl {
  /* matiz, saturação, luz — fácil de ajustar */
  background: hsl(265 70% 65%);
}

dica Prefira hsl() para montar paletas: muda só o primeiro número (o matiz) e você varre o arco-íris mantendo o mesmo "tom".

02

Sombras & profundidade

Uma box-shadow não é só "uma sombra embaixo". Empilhando várias você cria relevo, brilho (glow) e a sensação de camadas. Passe o mouse nos cartões.

Resultado · passe o mouse
suave
glow
camadas
CSS
/* x  y  desfoque  espalhamento  cor */
.soft {
  box-shadow: 0 18px 40px -12px rgba(0,0,0,.6);
}
.glow {
  box-shadow: 0 0 36px -4px #FF6B6B;
}
/* várias sombras separadas por vírgula */
.camadas {
  box-shadow:
    0 2px 0 #3a3170,
    0 4px 0 #322a64,
    0 16px 30px -10px rgba(0,0,0,.7);
}

dica O 4º valor (espalhamento) negativo "encolhe" a sombra — ótimo para sombras suaves e realistas, mais próximas da luz natural.

03

Tipografia viva

Texto também é design. Peso, espaçamento entre letras e até gradientes aplicados ao texto transformam palavras comuns em destaque.

Resultado
Tipografia
com personalidade
CSS
.big {
  font-weight: 800;
  letter-spacing: -0.04em;
  /* gradiente aplicado às letras */
  background: linear-gradient(100deg, #4ECDC4, #B79CFF);
  background-clip: text;
  color: transparent;
}
.spaced {
  letter-spacing: 0.5em;
  text-transform: uppercase;
}

dica Use em em letter-spacing: o valor acompanha o tamanho da fonte, então o espaçamento fica proporcional em qualquer tela.

04

Transições suaves

A transition é a forma mais barata de deixar uma interface viva: ela interpola entre dois estados (normal e :hover) ao longo do tempo. Passe o mouse.

Resultado · passe o mouse
CSS
.botao {
  transition:
    /* propriedade  duração  curva */
    transform 0.25s cubic-bezier(.22,1,.36,1),
    box-shadow 0.25s ease;
}
.botao:hover {
  transform: translateY(-3px) scale(1.04);
  box-shadow: 0 12px 28px -8px #FFC857;
}

dica Anime sempre transform e opacity em vez de top/width. Eles rodam na GPU e mantêm a animação a 60fps.

05

Transformações 2D e 3D

transform move, gira, escala e inclina elementos — inclusive no espaço 3D. Com perspective e rotateY dá pra virar um cartão. Passe o mouse no cartão.

Resultado · passe o mouse
Frente
Verso 👋
CSS
.flip { perspective: 900px; }
.flip-inner {
  transform-style: preserve-3d;
  transition: transform 0.7s;
}
.flip:hover .flip-inner {
  transform: rotateY(180deg);
}
.flip-face {
  backface-visibility: hidden;
}

dica backface-visibility: hidden é o segredo do flip: ele esconde o "verso" de cada face enquanto o cartão gira.

06

@keyframes & animações

Transições vão de A a B. Já a animation com @keyframes roda sozinha, em loop, passando por quantos estados você quiser — flutuar, pulsar, girar.

Resultado · roda em loop
CSS
/* 1. descreva os quadros */
@keyframes float {
  0%, 100% { transform: translateY(0); }
  50%      { transform: translateY(-22px); }
}
/* 2. aplique no elemento */
.orb {
  animation: float 3s ease-in-out infinite;
}
.spinner {
  animation: spin 1s linear infinite;
}

dica A palavra infinite faz repetir pra sempre; troque por um número (ex.: 3) para rodar só algumas vezes.

07

Layout com Grid

Esqueça float e margens mágicas. display: grid divide a tela em colunas e linhas, e cada item pode ocupar quantas células quiser.

Resultado
span 2 colunas span 2 linhas 3 4 5 6
CSS
.grid {
  display: grid;
  /* 3 colunas de tamanho igual */
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}
.wide { grid-column: span 2; }
.tall { grid-row:    span 2; }

dica A unidade fr ("fração") reparte o espaço livre. 1fr 2fr = a segunda coluna fica com o dobro da largura. Para alinhar itens em linha, use display: flex.

08

Variáveis CSS · playground

Variáveis (--minha-cor) guardam valores reutilizáveis. Mude uma só e tudo que depende dela se atualiza. Mexa nos controles e veja em tempo real.

Resultado · interativo
CSS
.box {
  /* defina as variáveis */
  --hue: 320;
  --radius: 24px;
  /* use com var() */
  border-radius: var(--radius);
  background: hsl(var(--hue) 80% 62%);
}
/* JS muda só a variável: */
// box.style.setProperty('--hue', 200)

dica Defina suas variáveis no seletor :root para usá-las no site inteiro — é a base de temas claro/escuro.

09

Recursos sofisticados

O CSS moderno vai muito além do básico: recortar formas com clip-path, criar vidro fosco com backdrop-filter e misturar camadas com mix-blend-mode.

Resultado
clip-path
glass
blend
CSS
/* recorta um pentágono */
.clip {
  clip-path: polygon(50% 0, 100% 38%, 82% 100%, 18% 100%, 0 38%);
}
/* vidro fosco (glassmorphism) */
.glass {
  background: rgba(255,255,255,.08);
  backdrop-filter: blur(8px);
}
.blend::after {
  mix-blend-mode: hard-light;
}

dica backdrop-filter embaça o que está atrás do elemento — diferente de filter, que afeta o próprio elemento.
